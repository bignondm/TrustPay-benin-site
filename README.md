🌐 TrustPay PME — Site Vitrine

Site web officiel de TrustPay PME, développé avec Next.js, présentant la solution de digitalisation des paiements et de gestion financière dédiée aux PME africaines.

🎯 Objectif du site

Ce site a pour but de :

Présenter la solution TrustPay PME
Expliquer les समस्यatiques des PME (informel, cash, comptabilité)
Mettre en avant les fonctionnalités clés
Convertir les visiteurs en utilisateurs
Rediriger vers l’application mobile
💡 À propos de TrustPay PME

TrustPay PME est une solution digitale qui permet aux entreprises de :

Encaisser des paiements via QR code
Générer des factures conformes
Automatiser leur comptabilité
Construire un historique financier fiable

👉 Objectif : rendre les PME structurées, crédibles et finançables

🚀 Installation
1. Cloner le projet
git clone https://github.com/ton-username/trustpay-website.git
cd trustpay-website
2. Installer les dépendances
npm install
3. Lancer le serveur
npm run dev
